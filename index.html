About

Hi there, I'm Ken Dunn. Welcome to my portfolio!

I'm a software engineer/web dev working my way into a new career. Previously, I was an audio engineer at a theater in Brooklyn, NY. I learned a lot of skills there and got to work on some very interesting shows. As I grew in skills, I realized that what I was most interested in was how the equipment that I was using worked. I decided that it was time to figure that out!
When I was accepted to General Assembly, I was very excited. Learning how to actually code was an interesting prospect. And after going through the course, I have learned so much and have realized that this what my career should be. The fact that I can think both logically and creatively each day in the programming environment is extremely exciting for me.



Personally, I enjoy board games and video games. I like to read and I like to watch to engaging TV shows. I also enjoy taking trips to my friends, celebrating the positive things in our lives, and playing spikeball. Also, I look forward to learning what it's like to be a dog dad since I will soon have a new dog in my life for the first time.

Personal Projects

Black-Jack-Black

This is Black-Jack-Black, a game I created with html, css, and javascript that allows you to play Blackjack against the computer. Bet some chips and see how much "money" you can make! And I hope you like Jack Black, because what would Blackjack be without Jack Black!

Disclaimer: This app does not use real money. You cannot gain nor lose money while using this app. It's still fun though, I promise.

Board Game Catalogue

This is an app that allows the user to manage lists of board games. It is a MERN stack application which utilizes some session and authorization logic as well. When you use this app, you will be able to add games to a few different lists of your choice; games you have, games you want to play, and games you have played. This lets the user keep track of what games they like and what kind of games they like as well!

Plant Buddi

Plant Buddi lets the user keep track of all their plants! This is also a MERN stack application that uses an external API to search for their plants, and keep track of different needs for each of their plants. This project was made with a small group of peers, which helped me to learn what a real world work environment would be like.

Zukies Dookies

Welcome to Zookies Dookies, an app to keep track of the health of your dogs. It was created with python/Flask, PostgreSQL, and React. The main features of this app are the ability to add different vaccines and medicines that your dog has been given, and to track your dogs' poops. This will let you give your vet all the information they will need on your visits with them.
